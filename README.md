# Signature ID 

Self soverign identity based dapp built on xdc


## To run in a local environment

### cloning
```
    git clone https://github.com/monishwarmc/SignatureID.git
```

### open frontend folder
```
  cd frontend/
```

###  install dependencies for the project
```
    yarn install
```
// or

```
    npm install
```


### run the project
```
    yarn dev
```

// or

```
    npm start dev
```
